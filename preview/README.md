# 网页预览控制台

这个目录提供一份不依赖 macOS 的网页预览，用来快速校验原生卡片 app 的布局和核心交互草模。
左侧区域只保留轻量的日期模拟和状态提示，不属于产品本体；真正的产品界面仍然只有卡片、倒计时总览和编辑倒计时。

## 推荐的本地预览方式
网页预览现在会直接读取仓库里的真实文案库和权重，所以更推荐在仓库根目录运行：

```bash
python3 -m http.server 8000
```

然后打开：

```text
http://localhost:8000/preview/
```

## 直接打开
- 双击 `index.html` 也能看
- 但浏览器可能拦住本地 `fetch`
- 这时会退回到内置的最小示例数据

## 它模拟了什么
- 多个倒计时页的卡组切换
- 当前日期的手动模拟与快捷跳转
- 原生偏好里最关键的透明度与整理动作映射
- 每天稳定随机的副标题
- 双击标题或日期牌编辑倒计时
- 倒计时总览的切换与找回
- 网页里的卡片拖动模拟
- 从页迹拖出一张单卡
- 点摘录卡右侧花朵后会立即换出今天的新一句
- Wendy 双语条目的显示方式
- 小彩蛋的掉落感

## 它不模拟什么
- 真正的 macOS 窗口层级与多窗口注册表
- 原生窗口拖回并组的接纳高亮与合并
- AppKit/SwiftUI 原生字体渲染
- 系统级窗口吸附与 Spaces 行为
