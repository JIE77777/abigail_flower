# 网页模拟预览

这个目录提供一份不依赖 macOS 的网页模拟版，用来预览原生卡片 app 的布局和气质。
它现在也能模拟多个独立倒计时页，并支持在卡片里直接编辑标签和目标日期。

## 推荐的本地预览方式
网页预览现在会直接读取仓库里的真实文案库和权重，所以更推荐在仓库根目录运行：

```bash
python3 -m http.server 8000
```

然后打开：

```text
http://localhost:8000/preview/
```

## 直接打开
- 双击 `index.html` 也能看
- 但浏览器可能拦住本地 `fetch`
- 这时会退回到内置的最小示例数据

## 它模拟了什么
- 多个独立倒计时页
- 当前日期的手动模拟与快捷跳转
- 每天稳定随机的副标题
- 双击标题或日期牌编辑当前倒计时页
- 点摘录卡右侧花朵后会立即换出今天的新一句
- Wendy 双语条目的显示方式
- 小彩蛋的掉落感

## 它不模拟什么
- 真正的 macOS 窗口行为
- AppKit/SwiftUI 原生字体渲染
- 系统级拖动与层级
