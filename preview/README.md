# 网页模拟预览

这个目录提供一份不依赖 macOS 的网页模拟版，用来预览原生卡片 app 的布局和气质。

## 直接打开
- 双击 `index.html`

## 更稳的本地预览方式
如果浏览器对本地文件有安全限制，可以在仓库根目录运行：

```bash
python3 -m http.server 8000
```

然后打开：

```text
http://localhost:8000/preview/
```

## 它模拟了什么
- 距离 8 月 31 日的倒计时
- 每天稳定随机的副标题
- 点花朵后会立即换出今天的新一句
- Wendy 双语条目的显示方式
- 小彩蛋的掉落感

## 它不模拟什么
- 真正的 macOS 窗口行为
- AppKit/SwiftUI 原生字体渲染
- 系统级拖动与层级
